# FastBox Delivery System

## Objective
This project simulates a one-day delivery operation.

## Features
- Read JSON input
- Assign nearest agent
- Calculate delivery distance
- Generate report
- Save report to JSON file

## Assumptions
1. One package delivered at a time
2. Agent return journey not included
3. First nearest agent selected during tie
4. Static agent positions

## How to Run

python main.py

## Output
report.json
